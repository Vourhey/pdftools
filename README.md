# PDF Tools

Usage:
```
usage: pdftools.py [-h] {split,s,merge,m} ...

PDF Toolkit by Vourhey

positional arguments:
  {split,s,merge,m}
    split (s)        Splits a given pdf into pages
    merge (m)        Merges given pdfs into one pdf file

optional arguments:
  -h, --help         show this help message and exit
```
